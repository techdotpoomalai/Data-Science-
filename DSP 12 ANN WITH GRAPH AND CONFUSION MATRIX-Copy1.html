model_1 = Sequential([
    
    Dense(units =128, input_dim =TOTAL_INPUTS, activation ='relu', name = 'm1_hidden1'),
    Dense(units =64,  activation ='relu', name = 'm1_hidden2'),
    Dense(units =16,  activation ='relu', name = 'm1_hidden3'),
    Dense(units =10, activation = 'softmax', name = 'm1_ouput')
    ])

model_1.compile(optimizer='adam', 
                loss='sparse_categorical_crossentropy', 
                metrics=['accuracy'])

Why do we use Softmax? The main advantage of using Softmax is the output probabilities range. The range will 0 to 1, and the sum of all the probabilities will be equal to one. If the softmax function used for multi-classification model it returns the probabilities of each class and the target class will have the high probability.

Adam is an adaptive learning rate method, which means, it computes individual learning rates for different parameters
